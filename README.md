# Instruções Iniciais

iniciar um repositorio git
```
git init
```

criar o arquivo
```
thouch arquivo.txt
```

editar o arquivo
```
vim arquivo.txt
```

preparar arquivo para commit
```
git add arquivo.txt
```

realizar commit commit
```
git commit -m "Primeiro Commit"
```

enviar arquivo para repositorio central
```
git push origin master
```
